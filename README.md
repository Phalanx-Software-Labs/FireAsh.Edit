# Mimic — Fire Ash mod installer (APK only)

This repo is **only** the Android installer APK. You do **not** get the source tree when you clone or download the default zip from GitHub — use **Releases** or the link below.

## Download

**[FireAshModInstaller-debug.apk](https://github.com/Phalanx-Software-Labs/Mimic/raw/main/FireAshModInstaller-debug.apk)**

Install on your device, grant folder access, pick the Pokémon Fire Ash folder that contains **`Data`**, then tap **Install bag-cheat mod**.

---

**Developers:** Keep your Android Studio project, Ruby mod sources, and `inject_mod.rb` workflow **outside** this repo (or in another repository). When you have a new APK, replace `FireAshModInstaller-debug.apk` at the root, commit, and push.

FireAsh.Edit for Windows

1. On GitHub open the "Releases" page for this project (not the green Code button).
2. Open the latest release that says "Windows" in the title.
3. Under "Assets", download FireAsh.Edit.exe.
4. Double-click it. You do not install Python.

If Windows shows a SmartScreen warning: More info → Run anyway (the app is not Microsoft-signed).

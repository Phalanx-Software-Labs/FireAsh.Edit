## FireAsh.Edit for Windows

### What to download

Download **FireAsh.Edit.exe** from the **Assets** section below (not the green “Code” button).

### How to run it

1. Double-click **FireAsh.Edit.exe**.
2. You do **not** need Python or any other software.

### If Windows warns you

You may see “Windows protected your PC” because the app is not signed by Microsoft. Choose **More info**, then **Run anyway**, if you trust this download.

### Game

Pokémon Fire Ash **3.6 Part 2.2** only. Use **Scan C: for game** or **Choose game folder** inside the app.
